사용자 분들을 위해 더 편하고 아름다운 서비스를 제공하기 위해 Treead를 크게 개선했습니다. 오늘 배포된 주요 변경사항을 안내드립니다.

## 새로운 사용자 인터페이스

### 현대적인 디자인으로 완전히 개선되었습니다

Treead의 전체 인터페이스를 SNS 스타일로 재설계했습니다. 더 이상 전통적인 블로그 형태가 아닌, 트위터나 인스타그램 같은 소셜 플랫폼의 편안함을 경험할 수 있습니다.

- 깔끔하고 직관적인 피드 레이아웃
- 더 나은 가독성을 위한 타이포그래피 개선
- 브랜드 컬러(초록색)를 활용한 일관된 디자인
- 모든 페이지에서 동일한 헤더와 네비게이션

### 모바일 사용자를 위한 최적화

스마트폰에서 Treead를 이용할 때 훨씬 편해졌습니다.

- 모바일 화면에 완벽하게 최적화된 레이아웃
- 하단에 고정된 네비게이션으로 언제든지 주요 메뉴 접근 가능
- 터치하기 쉬운 버튼 크기와 간격

## 글쓰기 환경 개선

### 빠른 글 작성 기능

홈 화면에서 바로 글을 쓸 수 있는 기능이 추가되었습니다.

- 홈 피드 맨 위에 글쓰기 상자가 표시됩니다
- "무슨 생각을 하고 계신가요?"라는 친숙한 메시지
- 클릭하면 전체 글 작성 편집기로 이동합니다

### 글 작성 편집기 최적화

글을 작성할 때 화면 활용도가 크게 개선되었습니다.

- 전체 화면을 활용하는 에디터로 더 넓은 작업 공간 제공
- 제목 입력창과 본문 편집기가 명확하게 구분
- 헤더는 항상 보이도록 고정되어 저장 버튼에 언제든 접근 가능

## 사진 업로드 개선

### 더 큰 용량의 사진을 업로드할 수 있습니다

아이폰으로 찍은 고해상도 사진도 이제 문제없이 올릴 수 있습니다.

- 이전: 약 1MB 제한
- 현재: 20MB까지 지원
- 최대 10MB 미만의 사진이라면 모두 가능

고화질 사진을 올릴 때 "파일이 너무 크다"는 오류를 더 이상 보지 않게 됩니다.

## 콘텐츠 표시 방식 개선

### 긴 글을 읽기 쉽게

피드에서 너무 긴 글이 화면을 차지하는 것을 개선했습니다.

- 300자 이상의 길이가 큰 글은 자동으로 축약 표시
- "더보기" 버튼을 통해 전체 내용을 볼 수 있습니다
- 부드러운 애니메이션으로 자연스러운 확장/축소

이렇게 하면 피드를 스크롤할 때 더 많은 글들을 볼 수 있습니다.

## 댓글 및 상호작용 개선

### 더 직관적인 댓글 작성

게시물 상세 페이지에서 댓글을 작성하는 방식이 더 편해졌습니다.

- 프로필 사진이 함께 표시되어 더 개인적인 느낌
- 글자 수 카운팅으로 남은 자리수 확인 가능
- 로그인 사용자만 댓글을 작성할 수 있습니다

## 뒤에서 일어나는 변화

### 이미지 자동 정리 시스템

Treead는 이제 저장 공간을 효율적으로 관리합니다.

- 삭제된 글에 포함된 사진들은 자동으로 제거
- 업로드했지만 게시하지 않은 사진은 24시간 후 자동 삭제
- 저장 공간을 아껴서 더 빠른 서비스 제공

## 알려진 사항

- 게시물의 높이가 제한되어 피드가 더 깔끔해 보이지만, "더보기" 버튼으로 전체 내용을 언제든 볼 수 있습니다
- 로그인 상태에 따라 보이는 버튼이 달라집니다. (로그인 후 더 많은 기능 이용 가능)

## 앞으로의 계획

지속적으로 사용자 분들의 피드백을 반영하여 Treead를 개선해나가겠습니다. 혹시 불편한 점이나 개선 제안이 있으시다면 언제든지 문의해주세요.
